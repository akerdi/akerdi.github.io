git push origin branch 出现error

git push origin V4 报错:

这里表明你的点是behind。

那就 git pull, 但是报错:

git pull === git fetch + git merge

这里的意思是要你指明remote 和branch: git pull origin V4

如果要将合并的点rebase 掉，git pull --rebase origin V4
